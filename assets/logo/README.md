# Logo Assets

This directory holds the LocalMind AI brand marks. These are **placeholders** — drop your real files here.

## Expected files

| File | Purpose |
| --- | --- |
| `logo.svg` | Primary vector logo (preferred, scalable) |
| `logo-mark.svg` | Icon-only mark (for the sidebar / favicon) |
| `logo-light.png` | Raster logo for light backgrounds |
| `logo-dark.png` | Raster logo for dark backgrounds |
| `favicon.ico` | Browser favicon |

## Brand guidance

- **Gradient tokens:** violet `#7c3aed` → indigo `#6366f1` → cyan `#22d3ee`.
- **Vibe:** dark, premium, glassmorphism (Cursor / Raycast / Linear / Notion / Arc).
- The wordmark uses the **Inter** typeface to match the app.
- Keep the icon legible at 24px (sidebar collapsed state).

Until real assets are added, the app renders a text/gradient wordmark as the brand.
