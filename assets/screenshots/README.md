# Screenshots

This directory holds product screenshots used in the README and pitch materials. These are **placeholders** — capture your own from the running app at http://localhost:3000.

## Expected files (referenced by the README)

| File | Screen |
| --- | --- |
| `dashboard.png` | Dashboard with live system stats + Ollama pill |
| `workspace.png` | AI Workspace text transforms |
| `documents.png` | Document Intelligence (upload + analyze) |
| `knowledge.png` | Knowledge Base RAG answer with sources |
| `search.png` | ⌘K command palette / smart search |
| `settings.png` | Settings (models, temperature, theme) |

## Capture tips

- Use the **dark theme** (default) for the premium look.
- Browser zoom ~110%, window width ~1440px.
- Include the green **Ollama online** status pill where visible — it reinforces the on-device story.
- Prefer PNG for crisp UI; keep files reasonably sized (< 1 MB each).
- For the pitch, a short screen recording of the "airplane-mode" moment (offline transform) is very effective.
